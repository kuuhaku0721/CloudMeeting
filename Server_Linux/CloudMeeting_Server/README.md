# CloudMeeting_Server
视频会议后台。涉及进程池，线程池，socket通信，unix域套接字。客户端地址：https://github.com/muleimulei/CloudMeeting

## 使用方法
``
usage: ./app [host] <port #> <#threads> <#processes>
``

#### threads 用于处理客户端连接的线程数
#### processes 开启的房间数

 QT 5.12.2(MSVC 2019, 64 bit)
